End-to-End Message Flow

File Structure

📡 Channels
src/cli/main.c

CLI REPL: readline, slash commands, response display. Boots all subsystems.

src/gateway/gateway.c
src/gateway/gateway.h

HTTP server. Endpoints: /, /api/chat, /api/status, /api/tasks, /api/tmux, /api/email

src/channels/email/email_channel.c
src/channels/email/email_channel.h

IMAP poll + SMTP reply via libcurl. ;UID=N URL fetch delivers body directly. Allowlist enforced on inbound and outbound.

🚌 Bus & Sessions
src/bus/bus.c / bus.h

Thread-safe inbound + outbound queues. Channels post inbound_msg_t; worker dequeues; dispatcher routes outbound.

src/bus/agent_worker.c

Single worker thread. Dequeues messages, looks up session, calls agent_chat(), posts reply to outbound queue.

src/session/session.c / session.h

Hash table: (channel:user) → agent_t. Factory-cloned from template. TTL expiry. Pre-creates cli:local at boot.

🧠 Agent
src/agent/agent.c / agent.h

Agentic loop: build messages → LLM → parse tool calls → execute → iterate. max_iterations + max_messages.

src/agent/tool_registry.c

Global tool table. Provides JSON schema to providers. Dispatches tool calls by name string.

🛠 Tools
src/tools/shell/shell_tool.c

shell_exec via tmux. Command allowlist. Output capture with timeout.

src/tools/file_ops/file_tool.c

file_read / file_write. Path allowlist enforcement.

src/tools/email/email_tool.c

send_email. Calls email_channel_is_allowed() before smtp_send(). Clear error if blocked.

src/tools/scheduler/scheduler.c

schedule_add / schedule_control. Background tasks with intervals.

src/tools/serial/serial_tool.c

serial_exec. Native termios, device.conf registry, ANSI stripping.

src/tools/skills/skills.c

Scans workspace/skills/*/SKILL.md into system prompt.

🏗 Core & Providers
src/core/dobby.h

Master header. Types, log, config. VERSION_STRING = 1.0.0.

src/core/types.c/h

result_t, inbound_msg_t, outbound_msg_t, response_pair_t. String utils (str_trim, str_starts_with).

src/core/config.c/h

INI parser. config_get / config_get_int. Reads dobby.conf.

src/providers/ollama/ & openai/

LLM adapters. Common provider_t interface. OpenAI-compatible for Groq, Anthropic, etc.

src/providers/registry.c

Auto-detect provider from API key / URL patterns.

🔒 Security, Memory & UX
src/security/allowlist.c/h

4 ACL types: COMMAND, PATH, ENDPOINT, EMAIL. Glob matching. Runtime add/remove with persistence to allowlist.conf.

src/memory/md_memory.c/h

Markdown memory in workspace/memory/. Persists across sessions.

src/history/history.c/h

Append-only HISTORY.md. Date-grouped. Never injected into LLM.

src/ux/slash_commands.c/h

CLI slash commands. Includes full /email suite.

src/ux/typing.c/h

Spinner. No background thread — advances on on_thinking() callback only.

Session Isolation

Session key
channel:user_id e.g. cli:local, http:192.168.1.10, email:user@gmail.com
Isolation
Each key gets its own agent_t cloned from template. Conversation history fully isolated.
Factory
agent_factory() in main.c clones provider, system prompt, settings. Passed to session_manager_create().
TTL expiry
Sessions expire after session_ttl seconds idle (default 3600). dobby.conf [agent] session_ttl.

Email Channel — Receive + Send

Inbound IMAP
Poll thread wakes every poll_intervals. UID SEARCH UNSEEN via CUSTOMREQUEST. Fetches imaps://host/INBOX/;UID=N (native URL — body to write callback directly). RFC 2822 parse → allowlist check → bus post with key email:<from>.
Outbound SMTP
Reply triggers email_send(). Allowlist checked. In-Reply-To / References threading headers. libcurl SMTP with TLS.
3 enforcement points
(1) process_uid() drops unknown senders. (2) email_send() blocks non-listed reply targets. (3) email_channel_send_direct() + tool pre-check gives clear error to agent.

CLI Email Commands

/email status
IMAP URL, SMTP, address, poll interval, last UID, last sender, running state
/email allowlist
Show all ACL patterns. States whether enforcement is active.
/email allow <addr>
Add pattern at runtime. Persists to allowlist.conf [email]. Supports globs: *@company.com
/email deny <addr>
Remove pattern at runtime. Persists to file.
/email send <addr>
Quick test email. Checks allowlist first — blocks with error if not listed.
